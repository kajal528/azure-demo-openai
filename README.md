# azure-demo-openai

Frontend for an AI Knowledge Assistant built with Vite, React, TypeScript, Tailwind CSS, and Axios.

## Features

- Upload documents to the backend API
- View uploaded documents
- Ask AI questions about a selected document

## Getting Started

```bash
npm install
npm run dev
```

The app expects the backend API at `http://localhost:8081/api` (see `src/services/api.ts`).

## Scripts

- `npm run dev` — start the development server
- `npm run build` — build for production
- `npm run preview` — preview the production build
